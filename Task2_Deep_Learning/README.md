# 🤖 Task 2: Deep Learning Model for Classification

🚀 Deep learning classification model built using TensorFlow and Keras.

---

# 🎯 Objective

The objective of this project is to build a neural network capable of performing classification using deep learning techniques.

---

# 📊 Dataset

Breast Cancer Dataset from Scikit-learn.

- Total Samples: 569
- Features: 30
- Target Classes: 2

---

# ⚙️ Steps Performed

- Loaded dataset
- Data preprocessing
- Train-test split
- Feature scaling
- Built neural network
- Trained model
- Evaluated accuracy
- Generated predictions
- Saved trained model

---

# 🧠 Deep Learning Architecture

- Input Layer
- Dense Hidden Layer (16 neurons)
- Dense Hidden Layer (8 neurons)
- Output Layer (Sigmoid activation)
