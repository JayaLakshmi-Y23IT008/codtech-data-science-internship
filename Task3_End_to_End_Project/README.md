# 🏠 Task 3: End-to-End Data Science Project

🚀 Complete machine learning workflow for house price prediction using Scikit-learn.

---

# 🎯 Objective

To build a complete machine learning pipeline capable of predicting house prices based on housing features.

---

# 📊 Dataset

California Housing Dataset from Scikit-learn.

- Total Samples: 20640
- Features: 8
- Target: House Price

---

# ⚙️ Steps Performed

- Loaded dataset
- Data exploration
- Data visualization
- Train-test split
- Feature scaling
- Model training
- Prediction
- Model evaluation
- Saved trained model

---

# 📈 Output

- Predicted house prices
- R2 score
- Prediction graph
- Saved model

---

# 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Joblib
